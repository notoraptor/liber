INTERFACE GRAPHIQUE
	Double-cliquer sur liber.jar pour lancer le programme !
LIGNE DE COMMANDE
	java -jar liber.jar
REMARQUE
	Si possible, le programme ajoute une icône dans la barre des tâches.
		Faites un clic droit sur cette icône et choisissez "Quitter Libersaurus" pour fermer définitivement le programme.
